## Installation
1. Install Node.JS
2. Run `npm i` within the `client` folder
3. Install Python
4. Run `python -m venv .venv` within the `server` folder, then also use the venv to install `pip install -r requirements.txt`
5. Can run `./start_client` and `./start_server` from this main folder to make it easy
